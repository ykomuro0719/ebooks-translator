# EBooks Translator

---

## Summary
インターネット上の翻訳機能を利用し、電子書籍を翻訳するスクリプトです

## Requirement
* Python 3.6~

## Supported Format
* epub

## Setup
```shell
git clone https://github.com/ykomuro0719/ebooks-translator.git
cd ebooks-translator
pip install -r requirements.txt
```

## Usage
```shell
python analize.py
> 変換対象のepubファイルのパスを入力してください:PATH_TO_EPUB

```
